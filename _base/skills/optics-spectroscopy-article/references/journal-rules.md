## Профиль и объём

- Оригинальные статьи физико-математического профиля; англоязычная версия
  «Optics and Spectroscopy» выходит одновременно.
- Статья НЕ должна быть опубликована или подана в другой журнал (иначе -
  объяснение в «Комментариях для редактора» при подаче).
- Объём: **до 20 стр. и 8 рис.**; «Краткие сообщения» - до 5 стр. и 2 рис.
- Обзоры - только по предварительному согласованию с редакцией.
- Рубрику выбрать из списка журнала; для метаповерхностей/ПНЧ подходят:
  «Оптика низкоразмерных структур, мезоструктур и метаматериалов» (11),
  «Нанофотоника» (18), «Плазмоника» (19), «Оптические материалы» (10).
- Статьи с нарушением правил возвращаются без рассмотрения; отказ редколлегии
  окончателен.

## Структура рукописи (обязательные элементы, в этом порядке)

1. Рубрика (строкой над названием).
2. Название - максимально конкретное и информативное; без аббревиатур.
3. © Полный список авторов: инициалы и фамилии (инициалы перед фамилией).
4. Место работы: организация, почтовый адрес с городом, индексом, страной;
   при нескольких организациях - надстрочные индексы у фамилий; e-mail
   ответственного автора со звёздочкой `*e-mail: ...`.
5. Аннотация: информативная и подробная (методы + главные результаты; какие
   вопросы поставлены и какие ответы получены). Без аббревиатур и ссылок.
6. Ключевые слова.
7. Основной текст с подзаголовками (введение, метод, результаты, обсуждение,
   выводы/заключение). Детали математики - в Приложение в конце.
8. СПИСОК ЛИТЕРАТУРЫ (именно такой заголовок).
9. Таблицы (после списка литературы).
10. Рисунки с подрисуночными подписями (в конце; каждый рисунок дополнительно
    отдельным файлом).

Порядок материалов в электронной версии: текст -> литература -> таблицы ->
рисунки с подписями.

## Текст

- Межстрочный интервал 1.5, кегль 12 pt (требование OJS-чек-листа).
- Единообразие терминов, обозначений, единиц; избегать сокращений, кроме
  общеупотребительных; каждое сокращение расшифровать при первом упоминании.
- Десятичная ТОЧКА, не запятая.
- Латинские буквы - курсивом, КРОМЕ: математических операторов (sin, exp,
  max), химических элементов (H2SO4), аббревиатур (HF), векторов.
- Векторы - жирным шрифтом, БЕЗ стрелки сверху.
- Числа с десятичным множителем - с центрированной точкой: 3 · 10^11.
- **Размерности и обозначения в тексте, таблицах и на рисунках - только на
  английском** (nm, eV, arb. units; НЕ «нм», «эВ»).

## Формулы

- Располагаются по месту в тексте; нумеруются только те, на которые есть
  ссылки; нумерация сквозная; нумерованные формулы - с красной строки.
- Простое (одиночные буквы/переменные, единицы, простые выражения типа
  a^2+b^2=c^2, H2SO4) - набирать ТЕКСТОМ, без внедрённых рамок.
- Выносные формулы - целиком одной рамкой редактора формул (Equation,
  MathType); не собирать формулу из кусков текст+рамка+таблица.
- В редакторе формул - общие настройки шрифтов/размеров, без ручных
  переопределений отдельных символов.

## Таблицы

- Нумерация арабскими цифрами в порядке упоминания; после номера - название.
- У всех граф - заголовки; графы разделены вертикальными линиями.
- Сокращения слов в таблицах не допускаются.
- Только средствами MS Word (не пробелами/табуляцией).

## Рисунки

- Понятные и чёткие; несколько частей под одним номером (1,a, 1,b) - общая
  подпись. На осях - величины и единицы (только английский). Не перегружать
  надписями: всё текстовое - в подрисуночную подпись; кривые нумеровать и
  расшифровывать в подписи.
- Тоновые: >=600 dpi, растровые (tif, jpg); убрать надписи с изображения.
- Графики/спектры/схемы: векторные форматы (wmf, eps, cdr, opj); в поле
  рисунка не должно быть сетки (кроме осциллограмм).
- Цвет возможен только в online-версии; печать ч/б - проверить читаемость
  кривых в градациях серого (лучше различать номерами/штрихами, не цветом).
- Каждый рисунок - отдельным исходным файлом дополнительно к вставке в текст.

## СПИСОК ЛИТЕРАТУРЫ

- Все ссылки нумеруются; в тексте строго по порядку, в квадратных скобках.
- Одна работа = один номер (нельзя две работы под одним номером и одну под
  двумя).
- Указываются ВСЕ соавторы с инициалами. DOI - при наличии. Список должен
  отражать современное состояние области и не быть избыточным.

Форматы (примеры из правил):

- Журнал: `Инициалы. Фамилии. Название журнала, том (выпуск), страница (год).
  DOI` - переводная версия в квадратных скобках следом.
  Пример: `О.В. Истомина, С.К. Евстропьев, Е.В. Колобкова, А.О. Трофимов.
  Опт. и спектр., 124 (6), 742 (2018). DOI: ... [O.V. Istomina, ... Opt.
  Spectr., 124 (6), 774 (2018). DOI: ...]`
- Книга: `Авторы. Название курсивом, изд. (Издательство, Место, Год), том.
  DOI` ; фрагмент - добавить `p. N` или `ch. N`.
- Конференция: `Авторы. In: Название сборника, ed. by ... (Изд-во, место,
  год), vol., p. . DOI`
- Диссертация: `Автор. Название. Автореф. канд./докт. дис. (институт, город,
  год). URL`
- Интернет: `Авторы. Название [Электронный ресурс]. URL: ...`

## Пакет подачи (через личный кабинет OJS)

1. Файл(ы) статьи целиком (текст+таблицы+рисунки с подписями) в DOC/DOCX/RTF
   или TEX, порядок материалов как выше.
2. PDF статьи в удобном для чтения виде.
3. Исходные файлы иллюстраций - каждый рисунок отдельным файлом.
4. Отдельный файл: данные об авторах (ФИО, e-mail, телефоны) + e-mail
   ответственного за переписку.
5. Лицензионный договор (бланк на сайте журнала).
6. Направление учреждения с указанием возможности открытой публикации
   (для работ, выполненных в организациях).
7. Отдельный файл для переводчиков: список специальных терминов с английскими
   эквивалентами + английские транскрипции фамилий авторов, название,
   аннотация и ключевые слова по-английски.
8. Список возможных рецензентов - не менее ТРЁХ: ФИО, место работы, e-mail.

Права: авторы передают право публикации на русском и перевода на английский;
остальные права сохраняются. Корректуру вернуть в течение 3 дней; оттиски не
высылаются (статьи в открытом доступе на сайте).

## Чек-лист перед сдачей пакета

- [ ] Объём <=20 стр., рисунков <=8 (или формат «Краткое сообщение»).
- [ ] Рубрика указана; название без аббревиатур; аннотация без ссылок.
- [ ] Титульный блок: ©, авторы, организации с адресами/индексами, *e-mail.
- [ ] Интервал 1.5, кегль 12; десятичные точки; курсив/жирный по правилам.
- [ ] Размерности и подписи на рисунках - только английские.
- [ ] Нумерованные формулы - в рамках редактора формул, сквозная нумерация.
- [ ] Таблицы - Word-таблицы с заголовками граф и вертикальными линиями.
- [ ] СПИСОК ЛИТЕРАТУРЫ: порядок цитирования, все соавторы, DOI, переводные
      версии в [];
- [ ] Порядок: текст -> литература -> таблицы -> рисунки с подписями.
- [ ] Отдельные файлы: рисунки, авторы, термины/транскрипции, рецензенты (>=3),
      лицензионный договор, направление учреждения.
- [ ] Материал не опубликован и не подан в другой журнал.
- [ ] Живые правила на journals.ioffe.ru перепроверены (могли обновиться).
