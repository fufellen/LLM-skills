# Реестр проверенной русской научной терминологии

Проверяй этот реестр до интернет-поиска. Не считай предложенный моделью
буквальный перевод термином. Добавляй запись сразу после проверки, чтобы
следующая задача могла использовать результат без повторного поиска.

## Правила проверки и записи

1. Ищи точную русскую форму в русскоязычных научных источниках: статьях,
   диссертациях, монографиях, стандартах или материалах профильных научных
   организаций.
2. Проверяй употребление по доступному полному тексту или официальной
   странице статьи. Поисковый фрагмент, агрегатор и машинный перевод сами по
   себе не подтверждают термин.
3. Для статуса «подтверждено» записывай русскую форму, английское
   соответствие, сокращение, предметную область, прямую ссылку на источник и
   дату проверки. Для спорного или нового термина по возможности найди два
   независимых источника.
4. Если точная русская форма не найдена, ставь статус «не подтверждено» и
   записывай безопасную редакционную замену. Не выдавай отсутствие результата
   поиска за доказательство отсутствия термина вообще.
5. Повторяй отрицательный поиск только при появлении нового источника, новой
   предметной области или необходимости дать точное русское полное название.
   При последующем подтверждении обнови статус, источник и дату, сохранив в
   примечании историю решения.
